# Flutter Samples

> A flutter application for calculating tip with great UI design.

Main Features Used :
- Beautiful UI Design
- Sliver List/Grid
- Sticky Header List
- Image Picker
- Data Fetching From Network
- ClipPath, Clippers and CanvasPaint
- Bloc Package
- Provider Package
- Future Builder
- SearchBar
- SQFLite Offline Database
- DatePicker and TimePicker
